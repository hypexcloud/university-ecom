## State Management Pseudo-code

```typescript
// Determine what to show
const hasMentoringAccess = ['business', 'infinity'].includes(planType)
const hasInfinityFeatures = planType === 'infinity'

// Render logic
{hasMentoringAccess && <SessionCard />}
{hasInfinityFeatures && <InfinityServicesCard />}
{planType === 'fast' && <UpgradeCTA />}

// Tab rendering
<Tab value="overview">Overview Content</Tab>
<Tab value="courses">Course Content</Tab>
{hasMentoringAccess && (
  <Tab value="mentoring">
    Mentoring Content
    {hasInfinityFeatures && <PriorityBadge />}
  </Tab>
)}
```

This visual guide shows exactly how the dashboard adapts to each plan type, making it clear for developers and stakeholders what features appear for each subscription tier.
